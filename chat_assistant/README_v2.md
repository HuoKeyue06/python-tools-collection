# Agent CLI v2.0
## 版本概述
相比于v1.0单文件脚本，v2.0完成了模块化重构，为后续Agent功能扩展奠定基础。
## 项目结构
my-first-project/
- src/
  - __init__.py
  - llm_client.py #智谱APT封装
  - tools.py      #工具注册表 + 两个示例工具
- ai-cli.py       #主程序
- .env            #智谱API密钥
- chat_history.json #聊天记录
- requirements.txt #依赖包
- README.md        #v1.0 文档
- README_v2.md     #v2.0 文档（本文档）
## 模块化重构（主要改进）
### 1. 模块化架构
- 将API调用逻辑抽离为'ZhipuClient'类
- 将工具逻辑抽离为'ToolRegistry'类
- 主文件'ai-cli.py'代码精简
### 2.工具系统雏形
- 实现了 'GetCurrentTime'和'Calculate'两个工具'
- 工具遵循统一接口：'name',''description',''function','schema','excute()'
- 支持手动执行工具（为下周FunctionCalling API做准备）
### 安全性增强
- 计算器使用'eval'函数，已对输入进行严格过滤，限制危险操作
- 工具执行包含异常捕获，防止工具执行错误导致程序崩溃
### Agent架构中的角色
┌─────────────────────────────────────────┐
│              用户 / 大模型                │
└─────────────────┬───────────────────────┘
                  │ "帮我计算 123 * 456"
                  ▼
┌─────────────────────────────────────────┐
│           ToolRegistry（工具调度中心）    │
│  - 知道所有可用的工具                     │
│  - 负责把请求转发给正确的工具              │
│  - 统一管理工具的"说明书"（Schema）        │
└─────────────────┬───────────────────────┘
                  │
        ┌─────────┴─────────┐
        ▼                   ▼
┌───────────────┐   ┌───────────────┐
│ GetCurrentTime│   │  Calculator   │
│     Tool      │   │     Tool      │
└───────────────┘   └───────────────┘
